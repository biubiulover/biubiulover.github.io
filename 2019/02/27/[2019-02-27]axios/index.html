axios

基于Promise的HTTP库，https://github.com/axios/axios

axios中文文档：http://www.axios-js.com/zh-cn/docs/

特性

  • 从浏览器中创建 XMLHttpRequest
  • 从 node.js 发出 http 请求
  • 支持 Promise API
  • 拦截请求和响应
  • 转换请求和响应数据
  • 取消请求
  • 自动转换JSON数据
  • 客户端支持防止 CSRF/XSRF
